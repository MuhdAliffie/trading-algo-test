You are working on my trading backtesting codebase. I already have or will generate trade result files from testing multiple strategies across Asia, Europe, and US stock markets.

Create one simple interactive dashboard that reads the trade results and visualizes the trading timeline.

Main goal:
Build an interactive dashboard where I can select one or multiple strategies using checkboxes, choose a timeline/date range, and see the trading performance and trade activity clearly.

The dashboard should show:

1. Money movements

* Show a line graph of portfolio value or cumulative capital over time.
* The line graph must support multiple selected strategies at the same time.
* Each selected strategy should appear as a separate line.
* The graph should update when I change the selected strategy, ticker, region, or date range.

2. Buy and sell time

* Show a timeline chart displaying when buy and sell events occurred.
* Buy signals should be clearly marked.
* Sell signals should be clearly marked.
* The timeline should update based on selected strategy, ticker, region, and date range.
* It should be easy to see when a stock was bought and when it was sold.

3. Duration held after buy

* Show a table listing each completed trade.
* Required columns:

  * Region
  * Ticker
  * Company name, if available
  * Strategy
  * Buy date
  * Buy price
  * Sell date
  * Sell price
  * Holding duration in days
  * Trade return
  * ROI percentage
  * Exit reason, if available

4. ROI after each sell

* Show a bar graph of ROI after each sell.
* Each bar should represent one completed trade.
* The x-axis should show trade number, ticker, or sell date.
* The y-axis should show ROI percentage.
* The graph should update when I change filters.

Dashboard filters:

* Region filter:

  * Asia
  * Europe
  * US
  * All regions
* Strategy filter using checkboxes:

  * Volume
  * MACD
  * LSTM-Inspired Bollinger Band Mean Reversion
  * Ichimoku Cloud
* Ticker filter:

  * Single ticker
  * Multiple tickers if practical
  * All tickers
* Date range filter:

  * Start date
  * End date
* Optional filter:

  * Only profitable trades
  * Only losing trades
  * Only active BUY signals
  * Only completed trades

* trade_return

* exit_reason

* roi_pct = ((sell_price - buy_price) / buy_price) * 100

If portfolio value over time is not available, create it from trade results using initial capital and cumulative trade returns. However, clearly document this assumption in comments.
